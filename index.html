Projects

Glasgow Club Nights - 2017/2018

Glasgow Club Nights was a project made in a team of 4. The website was made usign Djange, HTML, CSS and JavaScript and it was designed to inform users about the nightlife available in Glasgow. It featured the opening hours and a description of each club to help users choose the club that best suited their needs

PAS Pain Tracker

PAS Pain Tracker was another team project however it was created in a team of 5 for a client. This project aimed to create a mobile app which would allow users that suffered from chronic pain to record when and where they felt pain. This was designed so that users could keep a record of any pain experienced and this record could be used to more accurately describe the symptoms they experienced when they visited the doctor. This project was completed using Android Studio, coding mainly in Java as well as SQL for the backend Database.

How to Best Visualise Groups In Social Networks 2019-2020

This project was undertaken as my Fourth Year Dissertation. It involved performing extensive research into the previous work done on Graph Drawings and Community detection. A number of experimental stimuli were created using the LFR benchmark before an online experiment was performed using a custom made website. The website was created by myself using a mix of HTML, CSS and JavaScript with the results being stored in a Google Firestore database. From the data gathered from my experiment was used in a paper which will be publsihed in the ASONAM-2020 Graph Drawing Conference.
